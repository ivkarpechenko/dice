# 🎲 Arcane Dice Roller — Legendary Dice Simulator

A beautifully crafted, arcane-themed dice roller web application. Roll multiple dice types (d4 to d100), track your history, and enjoy the immersive dark fantasy interface. Fully responsive and optimized for both desktop and mobile browsers, with Telegram WebApp support.

---

## ✨ Features

- **8 Classic Dice Types** — d4, d6, d8, d10, d%, d12, d20, d100 (percentile die included)
- **Adjustable Quantity** — Roll from 1 up to 20 dice simultaneously
- **Animated Rolling Effect** — Visual feedback with pulsing dice and simulated rolls
- **Roll History** — Stores last 20 rolls with timestamps (local session)
- **One-Click Re-roll** — Quickly repeat the last roll
- **Telegram WebApp Ready** — Expands to full screen and integrates with Telegram’s WebApp API
- **Responsive & Dark Fantasy UI** — Crimson, gold and deep violet palette with elegant serif fonts

---

## 🛠️ Tech Stack

- **HTML5 / CSS3** — semantic markup, flexbox, grid, custom properties
- **JavaScript (Vanilla ES6)** — no external dependencies
- **Google Fonts** — *Cinzel* (headings) + *Crimson Pro* (body)
- **SVG Icons** — custom vector dice shapes for each die type
